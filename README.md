## 🦷 Dental - Dental Booking System

A complete dental appointment booking system built with **React.js**, **Tailwind CSS**, **Node.js**, and **Express**.

---

## 📁 Project Structure

```
DENTAL-NET/
├── Frontend/     # React + Tailwind CSS (UI)
└── Backend/      # Node + Express (API)
```

---

## 🚀 Features

- Dentist profile & schedule management  
- Patient appointment booking  
- Appointment list & status tracking  
- RESTful API integration  
- Scalable architecture  

---

## 🖥️ Frontend Setup (React + Tailwind CSS)

```bash
cd DENTAL-NET
cd Frontend
npm install
npm run dev
```

---

## 🌐 Backend Setup (Node.js + Express)

```bash
cd DENTAL-NET
cd Backend
npm install
```

Create a `.env` file:

```env
PORT=5000
MONGODB_URI=mongodb://localhost:27017/dental
```

Run the server:

```bash
npm start
```

---

## ⚙️ Tech Stack

- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (optional)  
- **Tools:** Axios, dotenv, Git, Postman
